## ByteRacer

![ByteRacer](screenshots/screenshot000.png "ByteRacer")

### Description

A fun racing game with a story!

### Controls

Keyboard/Mouse:

 - WASD/Arrow keys - Move
 - R - Restart/retry
 - Scroll - Move through main menu

### Building

Run CMake and build. CPM should pull in everything for you.
